# Tailor Management System (MCS Final Project)

## Project Overview
The **Tailor Management System** is an Android-based application developed as part of my **Master of Computer Science (MCS)** coursework.

-This project was created to help tailor shops in keeping track of client information, look We added all of the features listed below. 
Through the use of a mobile application, this project help tailor shops in effectively managing customer records, clothing measurements, orders, and delivery schedules.

-Measurements for various clothing designs (e.g., shirts, pants, suits, dresses) are stored and categorized for both male and female customers.
It tracks each order with a specific shelf number for easy storage and retrieval.

-It keeps track of delivery dates and notifies the tailor of the number of days left on each customer's order.
-It automatically generates and sends messages to customers for:
-Order confirmation
-Measurement confirmation
-Delivery reminders
-Completion notifications



##  Objectives
- Digitize tailor shop operations
- Maintain customer and measurement records
- Track orders and delivery schedules
- Provide a simple and user-friendly Android interface


##  Technology Stack
- **Platform:** Android
- **Language:** Java
- **Build System:** Gradle
- **IDE:** Android Studio
- **Database:** SQLite
- **UI Assets:** Custom icons and images
